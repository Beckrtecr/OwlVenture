# OwlVenture - Legend of the Sky

A pseudo-3D pixel art open-world adventure game.

## Controls
- **WASD / Arrows:** Move & Glide
- **SHIFT:** Dash / Talon Sword
- **SPACE:** Fly Up / Dive Bomb
- **Click:** Shoot Beak Projectile
- **M:** Toggle Map (not fully implemented in demo)

## How to Play
1. Open `index.html` in your web browser.
2. Explore the infinite procedural forest.
3. Fight the purple slime monsters.
4. Use SPACE to fly over obstacles!

## Features
- **Pseudo-3D Depth:** Experience true flight with Z-axis mechanics.
- **Parallax Backgrounds:** Multi-layered environment rendering.
- **Dynamic Combat:** Melee and Ranged attacks.

## Infinite Mode
The world is currently set to 3200x3200 pixels, but the procedural generation engine can handle infinite scrolling with minor tweaks to `GameScene.js`.

## Assets
- Owl Sprite: Generated by AI.
- Environment: Procedurally generated pixel textures (Programmer Art).

Enjoy the flight!
